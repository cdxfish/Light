# light
a simple python web framework

build for study python web development
##
这是一个为了学习Python web开发原理而写的框架。</br>
支持的功能有：</br>
（1）WSGI路由转发功能</br>
（2）一个简单的模板引擎</br>
（3）一个简单的ORM，支持MySQL数据库</br>
（4）支持重定向</br>
（5）下载文件</br>
（6）返回json数据</br>
##
依赖：</br>
werkzeug</br>
pymysql</br>
##
python版本：</br>
2.7 3.5</br>
